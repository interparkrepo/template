フォルダの設定を行います。ログファイルや設定ファイルなどの保存場所をデフォルトの状態から変更する場合は、下記フォームより設定を行ってください。
「ログファイル格納フォルダ」や「データファイル格納フォルダ」には個人情報を含む大切な情報が含まれる可能性があります。なるべくブラウザからアクセスの出来ない場所に移動することをおすすめします。

（注意）移動先のフォルダはあらかじめ作成しておいた場合は、フォルダの中には何も入れないでください。
設定変更を行なうと、自動的に古いフォルダに入っているファイルやフォルダが新しいフォルダへ移動し、古いフォルダが削除されます。
※現在のフォルダの中に新しいフォルダを作成することは出来ません。
