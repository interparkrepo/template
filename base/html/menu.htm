リンク先を変更すると正しく動作し無くなる場合があります。特別な理由がない限り変更しないでください。
アクセス権で、「システム管理者権限」にチェックを付けると、アカウント設定で「システム管理者」として登録されたユーザーのみがアクセスできるメニューとなります。ホームページ管理者がアクセスする必要のないメニューにはチェックを付けてください。

リンク先を何も入力しないで登録するとメニューグループのタイトルになります。メニューを独自に分類させたい場合に利用してください。
